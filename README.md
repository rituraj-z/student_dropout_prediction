# Student Dropout Prediction

A machine learning project that predicts student dropout risk using logistic regression.

## Features

- Age, Study Time, Number of Failures, Number of Absences
- Grade 1, Grade 2, Final Grade

## Model

- **Algorithm**: Logistic Regression
- **Accuracy**: 100%
- **Train/Test Split**: 80/20

## Requirements

```
pandas
matplotlib
seaborn
scikit-learn
```

## Usage

Run the Jupyter notebook to:
1. Load and preprocess student data
2. Train the model
3. Predict dropout risk
4. Visualize results with confusion matrix
